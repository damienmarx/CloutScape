Overview

CloutScape AIO is a comprehensive management system for RSPS (RuneScape Private Servers), featuring a sophisticated Discord bot and a web-based administration panel.

Discord Commands

Player Commands

  • !register <username> - Create a new RSPS account linked to your Discord.
  • !download - Get the latest game client download link.
  • !stats [@user] - View your or another player's game statistics.
  • !leaderboard - View the top players by GP balance.
  • !help - Show all available commands.

Admin Commands

  • !setup - Automatically configure the Discord server with required channels and roles.
  • !addgp <@user> <amount> - Add GP to a player's account.
  • !removegp <@user> <amount> - Remove GP from a player's account.
  • !ban <@user> [reason] - Ban a player from the game.
  • !unban <@user> - Unban a player.
  • !broadcast <message> - Send a server-wide announcement.

Web Interface

The web interface provides a real-time dashboard for monitoring server status and managing configurations.

  • Public Home: Overview of bot statistics.
  • Setup Guide: Step-by-step instructions for new administrators.
  • Admin Dashboard: Secure area to view all configured servers and their status.

Technical Details

The system is built using:

  • Backend: Python with Flask and Discord.py
  • RSPS Server: Java-based 317 Elvarg framework
  • Integration: Custom modules for gambling, PvP, and rewards
